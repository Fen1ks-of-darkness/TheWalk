## TheWalk
is a short single-player third-person survival-horror.


## About the game
You awake in an unfamiliar, abandoned place with no memory of how you got there. Your goal is simple: find a way out.


| Action | Control |
|--------|---------|
| Move | `WASD` |
| Look | `Mouse` |
| Run | `Shift` |
| Crouch | `Ctrl` |
| Jump | `Space` |
| Interact | `E` |
| Grab Ledge | `B` |
| Flashlight | `F` |
